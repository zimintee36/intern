Git and GitHub Cheat Sheet

Setup and Configuration
Set username globally: git config --global user.name "Your Name"
Set email globally: git config --global user.email "youremail@example.com"
Enable colored output: git config --global color.ui auto
List all configurations: git config --list

Repository Initialization and Cloning
Initialize a repository: git init
Clone a repository: git clone <repository_url>
Clone a specific branch: git clone --branch <branch_name> <repository_url>

Staging and Committing Changes
Add a file to staging: git add <file>
Add all changes: git add .
Check status: git status
Commit changes: git commit -m "Commit message"
Commit all changes directly: git commit -am "Commit message"

Branching and Merging
List branches: git branch
Create a new branch: git branch <branch_name>
Switch branches: git checkout <branch_name> or git switch <branch_name>
Merge a branch: git merge <branch_name>
Delete a branch: git branch -d <branch_name>

Viewing History and Differences
View commit history: git log
View a compact log: git log --oneline
Show changes between commits: git diff <commit1> <commit2>
Show changes in the working directory: git diff
Blame a file: git blame <file>

Remote Repositories
Add a remote: git remote add <name> <url>
Fetch changes: git fetch
Pull changes: git pull
Push changes: git push
Push a branch: git push origin <branch_name>

Undoing Changes
Unstage a file: git reset <file>
Discard changes in a file: git restore <file>
Reset to a specific commit: git reset --hard <commit>
Revert a commit: git revert <commit>

Stashing Changes
Stash changes: git stash
List stashes: git stash list
Apply and remove the latest stash: git stash pop


Tags
List tags: git tag
Create a tag: git tag <tag_name>
Create an annotated tag: git tag -a <tag_name> -m "Message"
